# 03-Juicy-Match-3
Project 03 for MSCH-C220 3/10/2021

# Description
Match 3 game. Move the animals to connect 3! Level 1, get as many points as you can in 60 seconds. Level 2, get 2000 points as fast as you can.

## Implementation
Godot_v3.2.3
